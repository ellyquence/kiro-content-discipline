## Gate 1: Strategic Alignment ✓

- [ ] Content brief approved and linked
- [ ] Target audience clearly defined
- [ ] Content goal aligns with business objectives
- [ ] Unique angle differentiates from competitors
- [ ] Fits content calendar and editorial themes

**Brief Link**: [URL]

---

## Gate 2: Content Quality ✓

### Structure
- [ ] Headline scores 35+/60 on criteria
- [ ] Introduction hooks reader in first 100 words
- [ ] H2/H3 structure is logical and scannable
- [ ] Each section delivers on its heading
- [ ] Transitions connect sections smoothly
- [ ] Conclusion summarizes and prompts action
- [ ] Single, clear CTA present

### Substance
- [ ] All claims supported with evidence
- [ ] Statistics are current (within 2 years)
- [ ] Sources are credible and linked
- [ ] Examples/case studies included
- [ ] Original insights or perspective provided
- [ ] Word count within target range (+/- 10%)

### Accuracy
- [ ] Facts verified against original sources
- [ ] Names, titles, companies spelled correctly
- [ ] Dates and numbers double-checked
- [ ] Quotes accurately attributed
- [ ] No outdated information

---

## Gate 3: Voice & Tone ✓

- [ ] Matches brand voice profile
- [ ] Tone appropriate for content type
- [ ] Consistent throughout piece
- [ ] No jarring style shifts
- [ ] Would sound natural if read aloud
- [ ] No off-brand words or jargon
- [ ] Personality comes through

---

## Gate 4: Readability ✓

### Metrics
- [ ] Flesch-Kincaid Grade Level: _____ (Target: ≤8)
- [ ] Flesch Reading Ease: _____ (Target: ≥60)
- [ ] Active voice percentage: _____ (Target: ≥90%)

### Writing Quality
- [ ] Average sentence length ≤20 words
- [ ] No paragraph exceeds 5 sentences
- [ ] Complex terms explained
- [ ] Varied sentence structure
- [ ] No run-on sentences
- [ ] Lists used appropriately
- [ ] White space sufficient

### Technical
- [ ] Spell check passed
- [ ] Grammar check passed
- [ ] No typos or errors
- [ ] Punctuation correct
- [ ] Formatting consistent

---

## Gate 5: SEO & E-E-A-T ✓

### On-Page SEO
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Secondary keywords distributed naturally
- [ ] Keyword density 1-2% (not stuffed)

### Meta Data
- [ ] Meta title: _____ chars (Target: 50-60)
- [ ] Meta description: _____ chars (Target: 150-160)
- [ ] Meta description includes keyword
- [ ] Meta description has CTA
- [ ] URL slug is clean and keyword-rich

### E-E-A-T Signals
- [ ] Author byline present
- [ ] Author bio with credentials
- [ ] Publication date visible
- [ ] Sources linked to authoritative sites
- [ ] Expert quotes included (if applicable)
- [ ] Experience demonstrated ("In our testing...")

### Links
- [ ] Internal links: _____ (Target: 3-5 minimum)
- [ ] External links: _____ (Target: 2-5)
- [ ] All links tested and working
- [ ] Anchor text is descriptive
- [ ] No broken links

---

## Gate 6: Media & Visuals ✓

### Featured Image
- [ ] High quality and relevant
- [ ] Properly sized for theme
- [ ] Compressed (<200KB)
- [ ] Alt text written (with keyword where natural)
- [ ] Social sharing dimensions correct

### Inline Images
- [ ] All images relevant and add value
- [ ] All images compressed
- [ ] All images have alt text
- [ ] Proper attribution if needed
- [ ] Screenshots current/not outdated

### Formatting
- [ ] Headers properly nested (H1 > H2 > H3)
- [ ] Lists formatted consistently
- [ ] Bold/italic used appropriately
- [ ] No orphaned headings
- [ ] Mobile preview looks good

---

## Gate 7: Technical & Schema ✓

- [ ] Schema markup added (Article/BlogPosting)
- [ ] Open Graph tags set
- [ ] Twitter Card tags set
- [ ] Canonical URL correct
- [ ] No duplicate content issues
- [ ] Page loads under 3 seconds
- [ ] Mobile responsive verified

---

## Gate 8: Final Review ✓

### Fresh Eyes Check
- [ ] Read aloud—sounds natural
- [ ] Someone unfamiliar reviewed
- [ ] No placeholder text remaining
- [ ] All [brackets] filled in
- [ ] No "TK" or "TODO" markers

### Pre-Flight
- [ ] Preview in CMS matches expected layout
- [ ] Social share preview looks correct
- [ ] Email notification ready (if applicable)
- [ ] Promotion plan documented
- [ ] Team notified of publish time
